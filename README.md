# Debraj Mukherjee — Filmmaker Portfolio

Cinematic portfolio site for filmmaker, editor, and colorist Debraj Mukherjee. Bilingual (EN/IT) with dark/light mode, embedded YouTube videos, and scroll-driven animations.

## Stack

- **Next.js 15** (App Router, TypeScript)
- **Tailwind CSS** + custom monochromatic design system
- **Framer Motion** (curtain reveals, letter animations, scroll fade-ins)
- **next-themes** (dark/light mode)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Connected to Vercel — pushes to `master` trigger automatic deploys.
